Структура HTML документа

Основные элементы структуры:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Заголовок страницы</title>
    </head>
    <body>
        Содержимое страницы
    </body>
</html>

Описание элементов:

  • <!DOCTYPE html> - объявление типа документа
  • <html> - корневой элемент
  • <head> - метаданные документа
  • <meta> - метаинформация
  • <title> - заголовок страницы
  • <body> - тело документа
